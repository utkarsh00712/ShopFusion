# React Admin Dashboard Template

This folder contains a standalone React admin dashboard template. It is not wired into the main NexCart application and is provided as a reference UI.

## Tech Stack
- React 18
- Material UI
- Nivo charts
- FullCalendar
- Redux Toolkit

## Running Locally
```bash
npm install
npm start
```

The template runs on port 5000 by default.

## Notes
- This template is separate from `NexCartFrontend`
- The NexCart admin UI is implemented in `NexCartFrontend/src/admin`
